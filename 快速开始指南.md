# 🚀 快速开始指南

欢迎使用基于 VS Code 的第二大脑系统！这个指南将帮助您在 10 分钟内开始使用。

## 第一步：安装扩展 (2 分钟)

VS Code 会自动提示安装推荐扩展，点击 "安装" 即可。

**最重要的两个扩展**：
1. **Markdown All in One** - Markdown 编辑支持
2. **Foam** - 双向链接和知识图谱

## 第二步：了解目录结构 (1 分钟)

```
SecondBrain/
├── PARA/                    # PARA 方法论系统
│   ├── 01_Projects/        # 有明确截止日期的项目
│   ├── 02_Areas/           # 需要持续维护的生活领域
│   ├── 03_Resources/       # 参考资料和模板
│   └── 04_Archive/         # 已完成或不再活跃的内容
└── ZK/                     # Zettelkasten 笔记系统
    ├── 01_Fleeting/        # 速记和临时想法
    ├── 02_Literature/      # 文献和阅读笔记
    └── 03_Permanent/       # 永久知识卡片
```

## 第三步：创建第一个笔记 (3 分钟)

### 方法一：使用代码片段（推荐）

**PARA 系统模板**：
- `/para-project` - 创建项目模板
- `/para-area` - 创建领域模板  
- `/para-resource` - 创建资源模板
- `/para-archive` - 创建归档模板

**ZK 系统模板**：
- `/zk-fleeting` - 创建速记笔记
- `/zk-literature` - 创建文献笔记
- `/zk-permanent` - 创建永久笔记
- `/zk-daily` - 创建日记模板

### 方法二：使用快捷任务

按 `Ctrl+Shift+P` 输入 "Tasks: Run Task"，选择：
- 🔍 创建日记 - 快速创建今日笔记
- 🚀 快速创建PARA项目 - 创建新项目
- 💾 备份到Git - 保存工作到版本控制

## 第四步：尝试核心功能 (4 分钟)

### 🔗 双向链接
- 输入 `[[` 开始创建链接
- 选择现有笔记或创建新笔记
- `Ctrl+Click` 跳转到链接笔记

### 🔍 快速搜索
- `Ctrl+P` 快速打开文件
- `Ctrl+Shift+F` 全局搜索内容
- 使用标签搜索：`tags: fleeting` 或 `type: project`

### 📊 知识图谱
- `Ctrl+Alt+G` 显示知识网络图
- 可视化笔记之间的连接关系

### 📝 任务管理
- `Ctrl+Shift+P` → "Tasks: Run Task" 查看所有可用任务
- 使用内置的项目和备份功能

## 💡 最佳实践

### PARA 系统使用建议
1. **项目管理**: 每个项目都有明确的成功标准和截止日期
2. **领域维护**: 定期回顾领域状态，及时调整关注点
3. **资源分类**: 按使用场景而非来源分类资源
4. **及时归档**: 完成的项目要及时归档并记录经验

### ZK 系统使用建议
1. **从速记开始**: 随时记录想法，不要担心质量
2. **文献笔记**: 阅读时记录关键观点和个人思考
3. **永久笔记**: 用自己的话重新表达概念，建立连接
4. **定期回顾**: 定期回顾旧笔记，发现新的连接

### 跨系统协作
- 在永久笔记中链接相关的项目和资源
- 在项目中引用知识卡片支持决策
- 用标签系统连接两个体系的内容

## 🆘 常见问题

**Q: 两套系统会不会太复杂？**
A: 可以从其中一套开始，比如先用 PARA 管理项目，再逐步加入 ZK

**Q: 模板生成的 frontmatter 有什么用？**
A: 这些元数据确保了与 Obsidian 和 Notion 的兼容性，支持未来迁移

**Q: 如何在两个系统间建立连接？**
A: 使用双向链接 `[[]]` 在项目中引用知识卡片，在笔记中链接相关项目

**Q: Git 备份有什么好处？**
A: 版本控制让你可以追踪知识的演化历程，永不丢失重要内容

## 🔄 迁移准备

本系统设计时考虑了跨平台兼容性：

- **Obsidian 迁移**: 所有 frontmatter 格式完全兼容
- **Notion 迁移**: 元数据结构可直接导入为数据库属性
- **标准 Markdown**: 任何支持 Markdown 的平台都能打开
